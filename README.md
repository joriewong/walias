# walias

A tool for appending alias to `.zshrc` and `.bashrc` files.

## Installation

```bash
npm install walias -g
```

## Usage

```bash
# command and alias are required arguments
walias <alias> <command>
```

## Example

```bash
walias gst 'git status'
# execute 'gst' will show git status information
gst
```
