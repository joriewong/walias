# walias

A tool for appending alias to `.zshrc`.

## Installation

```bash
npm install walias -g
```

## Usage

```bash
# command and alias are required arguments
walias <command> <alias>
```

## Example

```bash
walias 'git status' gst
# execute 'gst' will show git status information
gst
```
